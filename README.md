# Developer AI Tools & Specifications

Welcome to the **dev-ai-tools** repository. This repository houses the technical specifications, coding standards, and agentic AI skills designed to align development workflows and automate routine software engineering tasks.

It consists of two main pillars:

- **Tech Specs**: The technical guidelines, folder architectures, libraries, configurations, and boilerplate templates that set the standard for the projects.
- **Skills**: Self-contained executable runbooks and prompts that direct AI agents (like Antigravity) to write code and bootstrap features conforming to the Tech Specs.
