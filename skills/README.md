# AI Skills - Table of Contents

This directory contains the executable skills (or prompt runbooks) for AI agents to automate typical development tasks in the workspace.

---

## Supported Skills

### React

1. **[Configure Axios Client](./react/configure-axios/SKILL.md)**
   Automates setting up a standardized, pre-configured Axios instance with environment variables in the utility folder.

2. **[Create React API Client](./react/create-api-client/SKILL.md)**
   Automates generating generic API client hooks, CRUD operations, and corresponding MSW unit tests.

3. **[Scaffold React Application](./react/scaffold-app/SKILL.md)**
   Automates the end-to-end scaffolding of a premium React + Vite + TS + Material-UI + Redux + MSW + Storybook project.

4. **[Create API Mock (MSW)](./react/create-mock/SKILL.md)**
   Automates generating mutable mock database seeds, CRUD handlers, and registration setups in the mocks module.

5. **[Create React MUI Theme](./react/create-theme/SKILL.md)**
   Automates building and registering custom light/dark design token palettes and style overrides under assets/themes/.

6. **[Generate Translations](./react/generate-translations/SKILL.md)**
   Automates building translated JSON catalogs for specific target languages, creating namespace indices, and registering them globally.

### Orchestration / Setup

1. **[Install Dev AI Tools & Specs](./open-spec-install-dev-ai-tools/SKILL.md)**
   Automates cloning the standardized developer AI tools repository, copying specifications and agent skills into a local `.agent/` folder, and rewriting relative links.
